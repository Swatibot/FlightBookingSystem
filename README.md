# Flight Booking System ✈️

## Overview

The Flight Booking System is a Java-based application developed to simplify the flight reservation process. The system allows users to view available flight information, select their preferred seat type, make payments, and receive booking confirmation through a user-friendly graphical interface.

## Features

* View available flights
* Enter passenger details
* Select seat type (Window, Middle, or Aisle)
* Payment processing
* Booking confirmation
* Simple and interactive GUI using Java Swing

## Technologies Used

* Java
* Java Swing (GUI)
* Object-Oriented Programming (OOP)

## Project Structure

* Passenger Information Management
* Flight Selection Module
* Seat Selection Module
* Payment Processing Module
* Booking Confirmation Module

## How It Works

1. Enter passenger name.
2. Select a flight from the available options.
3. Choose a seat type.
4. Enter the payment amount.
5. Click on "Book Ticket".
6. Receive booking confirmation with ticket details.
